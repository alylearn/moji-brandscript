# MOJi 品宣工作台

MOJi 品宣团队内部使用的视频内容制作工具集，覆盖从选题到发布的完整工作流。

## 工作流

```
📋 脚本选题 → 🤖 AI生成方案 → ✅ 人工确认选题 → 📝 AI生成脚本 → 📊 脚本评分 → 🎬 视频制作 → 🎨 封面文案+配图 → ✍️ 标题+正文
```

## 文件说明

| 文件 | 用途 |
|---|---|
| `index.html` | 工作台首页，导航入口 |
| `moji_script.html` | 脚本工作流：选题方法论 + AI生成选题方案 + 脚本输出 |
| `moji_cover.html` | 封面方法论：文案公式、设计规范、叙事参考、AI生图指令 |
| `moji_copywriting.html` | 小红书文案：标题生成 + 正文生成 |
| `moji_score.html` | 脚本评分器：7维度打分，生成评分指令发给AI |
| `moji_benchmark.html` | 对标账号参考库：参考答案阅览室、无痛单词 |
| `moji_manual_v27.html` | 产品手册（上传给AI作为知识库，不在工作台内展示） |
| `shared.css` | 公共样式 |
| `images/` | 封面参考图等静态资源 |

## 使用方式

1. 通过 GitHub Pages 访问，浏览器打开即用
2. 挑一个 AI 工具（Claude / GPT / 豆包 / DeepSeek）建项目
3. 上传 `moji_manual_v27.html` 作为 AI 项目的知识库
4. 在工作台中按步骤操作，复制生成的 AI 指令，粘贴到 AI 对话中执行

工作台本身不需要上传给 AI——它是给人看的 UI，AI 拿到的只是工作台生成的指令文字。

## 部署

GitHub Pages，根目录部署，所有文件使用相对路径。
